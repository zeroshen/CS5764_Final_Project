Analysis of Google Play Store Apps

Introduction

Above visualization gives an analysis of the free and paid applications on the playstore. This is done by measuring the number of successful and unsuccessful applications in each category (free/ paid). For measuring the success of an application, a threshold of 1,000,000 downloads has been selected. The first insight that this visualization provides is that there is a higher chance of success for free applications than there is for paid applications. Hovering over each color in the pie chart provides further information about the number of applications. A tooltip has been employed for the same.
